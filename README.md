## 引脚分布

|    功能    |     GPIO      |
| :--------: | :-----------: |
|    SCK     |      41       |
|    MOSI    |      37       |
|    MISO    |  38(可不用)   |
|     CS     |      39       |
|     DC     |      40       |
| BL（屏幕） |  36(不能用)   |
| BL（键盘） |      16       |
|  五向摇杆  | 4、5、6、7、8 |
|   KEY_TX   |      15       |
|  UART_TX   |       1       |
|  UART_RX   |       2       |
